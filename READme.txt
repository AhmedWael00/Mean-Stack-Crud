Thank you for choosing me for this project! Below are the instructions to set up and run the application on your machine.

--- PREREQUISITES ---
Ensure you have the following installed:
1. Node.js
2. MongoDB (Make sure the MongoDB service is running)

--- STEP 1: SETUP BACKEND (API) ---
1. Open a terminal/command prompt.
2. Navigate to the 'backend' folder:
   cd backend
3. Install the required dependencies:
   npm install
4. Start the server:
   node server.js

(You should see: "CRUD Server running on http://localhost:3000" and "MongoDB Connected")

--- STEP 2: SETUP FRONTEND (ANGULAR) ---
1. Open a NEW terminal window.
2. Navigate to the 'frontend' folder:
   cd frontend
3. Install the required dependencies:
   npm install
4. Run the Angular application:
   ng serve -o

The application will automatically open in your browser at: http://localhost:4200

--- HOW TO USE ---
1. Add Task: Enter a title and description, then click "Add Task".
2. Edit Task: Click the orange "Edit" button next to any task. Update the details and click "Update Task".
3. Delete Task: Click the red "Delete" button to remove a task permanently.
